# ELJABR

Custom classes for working with algebraic expressions.

# DEPS

- [avtomat](https://github.com/Liebranca/avtomat)

# SETUP

```bash
cd  $ARPATH
git clone https://github.com/Liebranca/eljabr
eljabr/install.pl
eljabr/avto
```

That will build the project into `$ARPATH/lib/eljabr/`.
